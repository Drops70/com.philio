# Philio Z-Wave devices

This app adds support for Philio Z-Wave devices to Homey.


Supported devices:
* PSM02   motion sensor 4:1
* PST02-A motion sensor 4:1


Currently supported languages:
* English

#Compatibility
Homey version >= 0.9.2 (due to z-wave rewrite)
